# customer_reader_exercise
## Setup
To set up the database run
```
rake db:migrate
```
To remove the database run
```
rake db:remove
```
To reset the database (remove then set up) run
```
rake db:reset
```
To run the specs (tests) run
```
rake rspec
```
## Execution
To execute the app run
```
rake run
```
To start a console run
```
bin/console
```
